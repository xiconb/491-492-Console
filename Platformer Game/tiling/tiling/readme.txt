This is the platformer game code. It is based on the tiling tutorial based on Lazy Foo Productions. All textures are under fair use with the authors as mentioned below:
ukelele sound from bensound.

This platformer demo has one level, 3 cloud textures, 1 grass texture, 1 user sprite bmp and a Goal texture. For computer movement is done with the left and right arrows and jumping with the up arrow. The goal is to reach the goal.  

Below are the SDL libraries used for the configuration:
Windows:
SDL2
SDL2main
SDL2_image
SDL2_mixer
